works

Bias ML

A website designed to read text from a news article and determine if the content is biased or not. The front-end is built in react, the backend is built with Google Firebase, and the detection algorithm is built using Google's AutoML Library. I helped design the front-end for this project and set up the Auto ML data retrieval system using Google Firebase cloud functions. Additionally, I assembled the script that used Puppeteer to scrape the article's contents from the page. The data visualization was also set up by me using ChartJS. Winner of SD Hacks 2021!

A website that scrapes data off the official NASA mission status page and presents it in a simpler fasion.
